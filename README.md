# Tic-Tac-Toe

Unbeatable tic-tac-toe AI using minimax - with a C++/pybind11 backend, bitboard state, and ARM NEON intrinsics for parallel win detection.

This is overengineered for the sake of learning, I'm not insane :)

## Quick Start

```bash
pip install -e .
tic-tac-toe
```

You play as **X** — enter moves as `row col` (0-indexed, e.g. `1 2`). The AI plays **O**.
